# Online Investment Project (InvestLocal)

## Description
InvestLocal is a web-based application that allows users to explore and manage **local investment opportunities**. The project includes separate pages for general users and administrators, providing a simple interface for signing up, logging in, browsing opportunities, and managing investments. This project was built using **HTML, CSS, and JavaScript**.

## Features
- **User Registration and Login:** Secure access for users to manage their accounts.
- **Admin Panel:** Allows administrators to manage investment opportunities.
- **Opportunities Page:** Users can view available local investment opportunities with details.
- **Responsive Design:** HTML and CSS provide a user-friendly and visually appealing interface.
- **Interactivity:** JavaScript enhances user experience with dynamic content.

## Project Structure
- `index.html` – Main landing page  
- `signup.html` – User registration page  
- `login.html` – User login page  
- `admin.html` – Admin management page  
- `opportunities.html` – Listing of investment opportunities  
- `style.css` – Global styles for all pages  
- `script.js` – JavaScript functionality
